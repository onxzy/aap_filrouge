## Fonctionnement
Lancer le programme
```bash
$ ./display_avl.exe <filePath> <stepCount>
```
| Argument | Description | 
| - | - |
| filePath | Chemin vers la liste des prénoms |
| stepCount | Nombre d'étapes à afficher |

Exemple
```bash
$ ./display_avl.exe Prenoms_V2.txt 10
```
